Prerequisites

Use of the TLP file extensions require Tulip to be installed on the machine.

Export

The base URL for all exports is: http://connectomes.utah.edu/Rabbit/Export/REPORTTYPE/FORMAT/?query string

Reporttype

is what form the exported data should take. Allowed values are:

  • Network: The connectivity of individual cells
  • Motifs: The connectivity of cells sharing the same label
  • Morphology: The 3D structure of cells

Format

is what type of file to create. Allowed values are:

  • DOT: Graphviz format
  • TLP: Tulip format

Examples

Morphology example for structures 180, 476,and 514: http://connectomes.utah.edu/Rabbit/Export/Morphology/TLP?ids=180,476,514

The URL’s “ids” query parameter is a comma delimited list of structure IDs.

Network example for all neurons within two degrees of separation from structure 180: http://connectomes.utah.edu/Rabbit/Export/Network/TLP?ids=180&hops=2

The URL’s “hops” query parameter is the degree of seperation to include for the requested cells

Motifs example. Currently all cells are included in motif queries: http://connectomes.utah.edu/Rabbit/Export/Motifs/TLP/
